# P2P Pixel Battle Royale Web Game

PeerJS 기반 실시간 P2P 멀티플레이어 탑뷰 2D 배틀로얄 웹 게임입니다.  
클라이언트는 HTML, CSS, Vanilla JavaScript ES Modules로 구동되며, 방장이 서버/호스트 권한을 가져 모든 게임 로직(충돌, 투사체, 피격, 킬 판정)을 계산하고 게스트들에게 스냅샷을 브로드캐스트합니다.

---

## 🎮 조작법 (Controls)

*   **이동 (Move):** `W, A, S, D` 또는 `방향키 (Arrows Keys)` (대각선 속도 보정 완료)
*   **조준 (Aim):** `마우스 포인터 (Mouse Move)` (내 캐릭터 중심으로 무기 조준선 회전)
*   **공격 (Attack):** **자동 공격 (Auto-Attack)**
    *   입장 시 선택한 무기 고유의 공격 연사력(딜레이)에 맞춰, 마우스 조준 방향으로 무기가 자동 사용됩니다.
*   **승리 요건 (Victory Condition):** 배틀그라운드 아레나 안의 모든 적들을 물리치고 마지막 한 명이 되면 승리합니다.

## ⚔️ 선택 가능 무기 사양

1.  **검 (Sword):** 근접 부채꼴형 베기
    *   데미지: 24 | 공격 간격: 0.55초 | 사거리: 70px | 각도: 110도
2.  **도끼 (Axe):** 360도 원형 스핀 회전 공격
    *   데미지: 34 | 공격 간격: 0.90초 | 사거리: 58px | 각도: 360도
3.  **활 (Bow):** 원거리 투사체 (화살 발사)
    *   데미지: 38 | 공격 간격: 0.80초 | 사거리: 520px | 화살 속도: 520px/s (첫 피격 대상 1인 관통 소멸)
4.  **창 (Spear):** 중거리 직선 찌르기
    *   데미지: 30 | 공격 간격: 0.70초 | 사거리: 115px | 찌르기 폭: 22px
5.  **건틀릿 (Gauntlets):** 초근접 신속 연타
    *   데미지: 12 | 공격 간격: 0.22초 | 사거리: 45px | 각도: 80도

## 📡 PeerJS P2P 멀티플레이 구조

*   **방 생성:** 방장은 방 코드를 입력해 방을 생성합니다 (내부적으로 방 코드 중복 및 충돌을 막기 위해 네임스페이스 접두사 처리).
*   **방 입장:** 닉네임과 무기를 선택한 뒤, 방장과 똑같은 방 코드를 입력하고 "참여하기"를 누르면 대기방 없이 즉시 게임 서버에 연결되어 배틀그라운드에 즉시 스폰됩니다.
*   **연결 무결성:**
    *   방장(호스트)이 도중에 나가면, 접속한 모든 게스트들에게 연결 해제 경고 모달이 표시되며 로비로 안전하게 회송됩니다.
    *   게스트가 나가면 호스트 측에서 해당 캐릭터가 배틀그라운드에서 추방(탈락) 처리됩니다.

## 🛠️ 개발 및 로컬 실행 방법

이 프로젝트는 Vite를 기반으로 실행 및 빌드됩니다.

### 1단계: 패키지 의존성 설치
```bash
npm install
```

### 2단계: 로컬 개발 서버 실행
```bash
npm run dev
```
개발 서버가 구동되면 브라우저에서 `http://localhost:3000`으로 접속할 수 있습니다.

### 3단계: 프로덕션 빌드 및 스타트
```bash
npm run build
npm start
```

## 🧪 브라우저 다중 탭 테스트 가이드

이 게임은 완전히 클라이언트 단에서 P2P 브라우저 직접 네트워크 기술로 진행되므로, 하나의 PC 및 동일 브라우저에서 여러 개의 탭이나 시크릿 창을 띄워 아주 손쉽게 테스트할 수 있습니다.

1.  **첫 번째 탭 (방장 생성):**
    *   접속 후, 닉네임 `HOST` 및 무기 `Sword`를 선택합니다.
    *   방 만들기 칸에 `PLAY77` 같은 고유 코드를 입력하고 **Host Game** 버튼을 누릅니다.
    *   즉시 맵 중앙 인근에 스폰되어 아레나가 시작됩니다.
2.  **두 번째 및 세 번째 탭 (게스트 참여):**
    *   새 탭 또는 시크릿 모드 브라우저를 엽니다.
    *   닉네임 `GUEST1`, `GUEST2` 등을 입력하고, 연타 속도가 빠른 `Gauntlet` 이나 원거리 `Bow` 등의 다른 무기를 골라봅니다.
    *   참여하기 칸에 방장이 활성화한 코드인 `PLAY77`을 똑같이 입력하고 **Join Game** 버튼을 누릅니다.
    *   연결이 수립되는 순간 즉시 방장의 전 화면 실시간 동기화 데이터가 전송되며, 아레나의 서로 다른 안전 거리에 스폰되어 전투에 동참하게 됩니다!

## ⚠️ 기술 제한 사항 및 안심 수칙

1.  **PeerJS 공식 퍼블릭 클라우드 브로커 사용:**
    *   이 앱은 별도의 시그널링 서버 설치 부담을 덜기 위해 default PeerJS 공식 무료 클라우드 브로커 서버를 타고 위상 정렬을 진행합니다.
    *   네트워크의 NAT 환경/방화벽 제약사항(일부 대학망, 기업 사무실망 인트라넷 등)에 따라 간혹 직접 홀펀칭 연결이 실패하여 TURN/STUN 리액션 실패가 발생할 수 있습니다. 이 과정에서는 표준 LTE/5G 모바일 핫스팟 또는 표준 가정용 Wi-Fi 망 환경에서 연결하는 것을 권장합니다.
2.  **보간 보정 (Linear Interpolation):**
    *   호스트는 초당 약 22회(22Hz) 전체 동기화 본체를 발송하며, 각 클라이언트는 네트워크 지터에 따른 끊김을 방지하기 위해 0.3 계수의 **선형 좌표 보간(lerp)**을 거쳐 부드럽게 글라이딩 렌더링을 진행합니다.
