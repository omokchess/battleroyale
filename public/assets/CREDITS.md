# Asset credits

## Ninja Adventure Asset Pack — Pixel Crushers (CC0 / public domain)

Used throughout the game world (characters, weapons, FX, tiles, status icons)
and, for the meta UI, the **Theme Wood** UI set under
`ninja/ui/Theme/Theme Wood/` (nine_path panels, buttons, tabs, slots, etc.).

- Source: Ninja Adventure Asset Pack
- License: CC0 1.0 Universal (public domain dedication) — free for any use,
  no attribution required (recorded here as a courtesy).
- Scope in this project:
  - Game world (Task 4): character body sheets, weapon sprites, slash/elemental
    FX sheets, tilesets, status-effect icons.
  - Meta UI (lobby / hub re-skin): the Theme Wood palette and frame language are
    the single visual reference for the medieval menu board. Where a component
    has no pack sprite (resource bar, notice strip, hub module cards), it is
    built in CSS using the pack's palette and outline weight — **no other UI pack
    is mixed in** (Task 4 cohesion rule).
